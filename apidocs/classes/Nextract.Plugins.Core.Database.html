Defined in plugins/core/Database.js:58

Raw query interface for select statements

Parameters:

  • dbName String

    A database name that matches a object key defined in your Nextract config file

  • sql String

    SQL statement to execute. Can be a fully formed SQL statement or a parameterized one with "?" placeholders. If the later, then sqlParams must be an array of values to be replaced in order.

  • sqlParams Array

    (optional) List of params to be subbed as a parameterized query

Returns:

Promise:

Promise resolved with an array of database rows that match the given select statement

Example:

ETL.Plugins.Core.Database.select('dbname', 'select * from tablename');
