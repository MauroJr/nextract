Defined in build/plugins/core/Output.js:95

Used to output files

Parameters:

  • fileType String

    Type of file to write; json, csv, or excel

  • data Array

    An array of objects to be wrtten to the file

  • filePath String

    Full path of file to write (include filename and extension)

  • formattingConfig Object

    Object contain config options for the file type being written.

    1. If cvs - all options allowed by cvs-stringify (http://csv.adaltas.com/stringify/) are supported
    2. If json - the only option is formatting with X number of spaces (e.g.) {spaces: 2}
    3. If excel - see https://www.npmjs.com/package/excel4node

Example:

ETL.Plugins.Core.Output.writeFile('csv', "path/to/file.csv", { header: true }, data);
