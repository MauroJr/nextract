String

Defined in build/plugins/core/Output/Output.js:37

Converts stream objects to csv strings (usually paired with toFile).

Parameters:

  • formattingConfig Object

    Object contain config options for the file type being written. All options allowed by cvs-stringify (http://csv.adaltas.com/stringify/) are supported.

Returns:

String:

Returns a string formatted as a CSV

Example:

var formattingConfig = { headers: true, columns: { first_name: 'first_name', last_name: 'last_name', ... } };
transform.Plugins.Core.Output.toCsvString(formattingConfig);

Converts stream objects to JSON strings (usually paired with toFile).

Parameters:

  • wrapJsonArray Boolean

    (defaults to true)

  • open String

    Custom opening string placed before JSON array. Defaults to '{\n\t"data": [\n\t'.

  • close String

    Custom close string placed after JSON array. Defaults to ',\n\t'.

  • seperator String

    Custom seperator places between array object elements. Defaults to '\n\t]\n}\n'.

Returns:

String:

Returns a string formatted as JSON

Example:

transform.Plugins.Core.Output.toJsonString(true);
