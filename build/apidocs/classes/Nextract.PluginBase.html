buildStreamTransform

(
  • streamFunction
  • streamFunctionType
)
Function

Defined in build/plugins/pluginBase.js:150

Accepts a stream transform function that conforms to the through2.obj stream wrapper API. Caller can choose to use through2 or one of the through2 helper modules (map, filter, spy). For more information see - https://github.com/rvagg/through2.

Parameters:

  • streamFunction Function

    Function that conforms to the through2.obj stream wrapper API

  • streamFunctionType String

    (optional, defaults to standard through2) standard, filter, map, or spy

Returns:

Function:

Returns stream transform wrapped using through2

Example:

var streamFunction = function(element, index) { return element.foo <= 10 };
return pluginName.runTask('sometaskname', streamFunction, 'filter');

runInWorker

(
  • workerMsg
)
Promise

Defined in build/plugins/pluginBase.js:102

Convinence method used by plugins to run code in a background web worker thread. A worker script must exist in the same directory as the plugin itself with a name of Worker.js. The format of the worker script must match the one defined by the npm workerjs module - https://www.npmjs.com/package/workerjs#node-mode---allowing-require

Parameters:

  • workerMsg Object

    The message to be passed to the worker (can be an object)

Returns:

Promise:

Promise resolved with worker response msg

Example:

var workerMsg = { cmd: 'orderBy', args: [collection, iteratees, orders] };
return pluginUtils.runInWorker(workerMsg);
